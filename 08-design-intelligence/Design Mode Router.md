# Design Mode Router

Date: 2026-04-24

## Mục Đích

File này quyết định khi nào dùng:

- `Flatsome Default`
- `Flatsome + Design Assist`
- `Flatsome + Directed Creative`

## Mode 1: Flatsome Default

### Dùng khi

- brief bình thường
- user không yêu cầu sáng tạo
- cần nhanh và ổn định
- page type quen thuộc

### Kết quả

- bám `111 Full Page`
- bám `Flatsome Native Core`
- ưu tiên pattern an toàn nhưng vẫn phải qua novelty gate

## Mode 2: Design Assist

### Dùng khi

- user muốn nhìn chuyên nghiệp hơn
- user muốn có đề xuất
- page trước bị chê lặp
- cần 2-3 hướng để chọn

### Kết quả

- dùng thêm:
  - `Style Taxonomy.md`
  - `Pattern Decision Matrix.md`
  - `Typography Mood Matrix.md`
  - `Spacing Rhythm Matrix.md`
  - `Motion Profile Matrix.md`

## Mode 3: Directed Creative

### Dùng khi

- user yêu cầu phong cách cụ thể
- user nói rõ muốn sáng tạo mạnh
- page là campaign, premium landing, homepage showcase

### Kết quả

- bật đầy đủ lớp `Design Intelligence`
- nhưng vẫn phải dịch về `Flatsome Native`

## Từ Khóa Gợi Ý Router

- `mặc định`
  - route về `Flatsome Default`
- `sáng tạo hơn`
  - route về `Design Assist`
- `theo phong cách Apple / Fluent / editorial / luxury / startup`
  - route về `Directed Creative`

## Quy Tắc Chọn Nhanh

1. Không có yêu cầu sáng tạo rõ:
   - `Flatsome Default`
2. Có chê trang cũ lặp hoặc muốn đẹp hơn:
   - `Design Assist`
3. Có phong cách chỉ định:
   - `Directed Creative`

## Kết Luận

Mode router giúp công ty không dùng sáng tạo quá tay cho mọi task, nhưng cũng không bị kẹt ở tư duy mặc định mãi.

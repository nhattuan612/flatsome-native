# Thư Mục Report Ngoại Lệ

Date: 2026-04-24

## Mục Đích

Đây là vị trí lưu chuẩn cho mọi `report + file/log` của `ngoại lệ hiếm` trong hệ Flatsome Native.

## Quy Tắc Đặt File

- mọi file phải theo mẫu: `EX-YYYY-MM-DD-XX` + phần mở rộng .md
- ví dụ: `EX-2026-04-24-01` + phần mở rộng .md

## Ghi Nhớ

- không lưu ngoại lệ rải rác ở chỗ khác
- nếu có ngoại lệ hiếm nhưng không có file tại đây thì ngoại lệ đó được xem là chưa hợp lệ
